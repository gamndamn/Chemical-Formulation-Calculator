# Chemical Formulation & Solubility Calculator

### This B2B-oriented R&D tool automatically calculates required component and solvent weights based on user-defined target concentrations. Additionally, the system provides real-time audits for dynamic solubility thresholds and safety limit compliance.

### Features:
- **Precise Manual Input:** Keyboard entry for exact concentration percentages.
- **Automated Safety Audits:** Immediate alerts when maximum safe limits are exceeded.
- **Dynamic Solvent Mapping:** Solubility thresholds automatically adapt based on the chosen solvent (Water, Ethanol, or Mineral Oil), preventing crystallization and precipitation before lab tests begin.
- **Interactive Visualization:** Clean data presentation powered by Streamlit.

### How to run locally:
1. `pip install streamlit pandas`
2. `streamlit run formulation_app.py`
